## How do I log into the Endform dashboard?

Go to the Vercel integration settings page for your installation and click **Open in Endform** in the top-right corner.

## What code hosting platforms are supported?

The Vercel integration supports [GitHub](/docs/guides/getting-started). If you use another CI/CD provider, follow the [getting started guide for other CI/CD providers](/docs/guides/getting-started/other-ci/existing/vercel).

## Can I run endform locally?

Yes. Run the Endform CLI in the standard way:

```sh
npx endform@latest test
```

This supports tests against both local dev servers and remote servers. See the [local CLI getting started guide for local dev servers](/docs/guides/getting-started/local-cli/existing/local) or the [local CLI getting started guide for remote servers](/docs/guides/getting-started/local-cli/existing/remote).
