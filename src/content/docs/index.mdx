Endform is a managed, distributed Playwright test runner and analytics dashboard.

Write your browser-based end-to-end tests using the [open source Playwright framework](https://github.com/microsoft/playwright).

Then you can run your suite on Endform's fully isolated cloud runners.
We run one test per virtual machine for maximum performance.

Endform also helps you maintain your suite over time by providing analytics in our [dashboard](https://endform.dev/app).

### Running Endform

To run tests with Endform, you only need to swap out the `playwright` CLI for the `endform` CLI. We support the same functionality as Playwright.

```sh
npx endform test
```
